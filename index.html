Problem

Nə üçün ADDIM lazımdır?

Azərbaycanda 600.000-dən çox vətəndaş üç əsas problemlə üzləşir. Bu problemlər onların müstəqil həyat sürmələrini maneə törədir.

01
📊

Məlumat Yoxluğu

Əlçatan məlumatın kəskin çatışmazlığı mövcuddur. İnklüziv dizayn normlarına uyğun rəqəmsal resurslar demək olar ki, yoxdur. Vətəndaşlar hüquqlarını, xidmətlərini və imkanlarını bilmir.

02
🚧

Fiziki Maneələr

İnfrastruktur universal dizayn standartlarına uyğun deyil. Piyada yolları, keçidlər və ictimai nəqliyyat hərəkət məhdudiyyətli şəxslər üçün əlçatan deyil. Şəhər mühiti istisnaedici xarakter daşıyır.

03
🔗

Sosial İzolyasiya

Sosial və bürokratik maneələr, cəmiyyətdə mövcud olan stereotiplər və yanlış yanaşmalar vətəndaşları izolə edir. Bərabər imkanların olmaması dərin sosial bərabərsizliyə yol açır.

Həll

ADDIM nədir?

ADDIM — əlilliyi olan şəxslərin sosial müdafiəsinin rəqəmsallaşdırılması və "müstəqil həyat" konsepsiyasının tətbiqi üçün nəzərdə tutulmuş çoxfunksiyalı texnoloji həlldir. Süni intellekt, Böyük Verilənlər və real-vaxt texnologiyaları əsasında qurulmuşdur.

Modul 01
🗺️

Əlçatanlıq Xəritəsi

Şəhər infrastrukturunun inklüzivlik səviyyəsini Böyük Verilənlər və real vaxt rejimində istifadəçi rəylərini təhlil edən rəqəmsal ekosistem. Hər məkanın əlçatanlıq dərəcəsi anlıq olaraq görünür.

Modul 02
📋

Xidmət Bələdçisi

Sosial müavinətlər və hüquqi prosedurlar üçün fərdiləşdirilmiş alqoritmik rəhbərlik. Hər vətəndaşa öz vəziyyətinə uyğun addım-addım istiqamət verir.

Modul 03
🤝

Sosial Könüllülük

Ehtiyac sahibi ilə ən yaxın məsafədəki resursun (könüllünün) intellektual eşləşdirilməsi. İnsan mərkəzli köməklik şəbəkəsi yaradır.

Modul 04
💼

Əmək Bazarı

Fiziki imkanlara uyğunlaşdırılmış məşğulluq modellərinin və distant iş imkanlarının aqreqasiyası. Hər kəsin potensialına uyğun iş imkanları.

Kontekst

Xarici və Yerli Təcrübə

🌍 Xarici Uğur Nümunələri Qlobal

  • AccessNow (ABŞ) — real vaxt əlçatanlıq xəritəsi ilə 50.000+ istifadəçiyə çatmış platforma. Crowd-sourced data modeli ADDIM-ə ilham mənbəyidir.
  • BE MY EYES (Danimarka) — görmə qüsurlu şəxslər üçün könüllü şəbəkəsi. 500.000+ aktiv könüllü ilə dünya miqyasında uğur qazanmışdır.
  • Wheelmap (Almaniya) — OpenStreetMap əsaslı əlil arabalı istifadəçilər üçün global xəritə platforması. 200K+ nöqtə məlumatı mövcuddur.

🇦🇿 Yerli Vəziyyət Azərbaycan

  • ƏƏSMN — Əlilliyi olan şəxslərə sosial xidmətlər göstərir, lakin rəqəmsal platforma mövcudiyyəti məhduddur. İnteqrasiya potensialı yüksəkdir.
  • Şəffaf Hökumət Portalı — Məlumat açıqlığı mövcuddur, lakin WCAG standartları tam tətbiq edilməyib. ADDIM bu boşluğu doldurur.
  • Dövlət İmtahan Mərkəzi — Əlil abituriyentlər üçün şərait yaradılır, lakin proseslər haqqında əvvəlcədən məlumat yetərli deyil.

  • İnstitusional dialoq prosesi

  • Strateji hesabatlılıq sistemi

Tövsiyələr

Strateji Təkliflər

1

Dövlət-Özəl Tərəfdaşlıq (PPP) Modeli

ƏƏSMN və Bakı Nəqliyyat Agentliyi ilə inteqrasiya — dövlət resursları ilə texnoloji innovasiyanın birləşməsi.

2

WCAG 2.1 AA Standartları

Ekran oxuyucular, kontrast nisbətləri, klaviatura naviqasiyası — hər istifadəçi üçün tam əlçatanlıq.

3

Açıq Məlumat (Open Data) Strategiyası

İctimai binaların əlçatanlıq məlumatları API vasitəsilə paylaşılsın — şəffaf, ictimai faydalı ekosistem.

4

Pilot Layihə

Bakı Metropoliteni + İçərişəhər + 3 dövlət xidməti ofisi — real dünya sınağı ilə başlanğıc.

5

KSM Çərçivəsində Korporativ Əməkdaşlıq

Korporativ Sosial Məsuliyyət (KSM) çərçivəsində 50+ şirkətlə əməkdaşlıq — biznes üçün dəyər, cəmiyyət üçün fayda.

Analiz

SWOT Analizi

Platformanın güclü tərəflərini, zəifliklərini, imkanlarını və təhdidlərini obyektiv qiymətləndirməsi.

💪
S — Güclü Tərəflər
Strengths
  • Azərbaycanda ilk hərtərəfli rəqəmsal əlçatanlıq platforması
  • Süni intellekt və Big Data inteqrasiyası
  • Dörd modul: Xəritə, Bələdçi, Könüllülük, İş Bazarı
  • Dünya təcrübəsinə əsaslanan arxitektura
  • 600.000+ potensial istifadəçi — böyük bazar
⚠️
W — Zəif Tərəflər
Weaknesses
  • Azərbaycan dili üçün nitq tanıma sisteminin zəifliyi
  • MVP mərhələsi üçün kapital tələbi
  • İnfrastruktur auditinin mürəkkəbliyi
  • Texnofobiya səbəbilə yaşlı istifadəçilərdə adaptasiya çətinliyi
  • İnternet kəsilmələrinə həssaslıq
🚀
O — İmkanlar
Opportunities
  • ƏƏSMN və BNA ilə dövlət-özəl tərəfdaşlığı (PPP)
  • Azərbaycanda rəqəmsallaşma strategiyasının genişləndirilməsi
  • 50+ korporativ tərəfdaş və KSM maliyyələşməsi
  • Regional ekspansiya — Naxçıvan və digər şəhərlər
  • Beynəlxalq qrant və EU Accessibility Act uyğunluğu
🛡️
T — Təhdidlər
Threats
  • Bürokrasiya və dövlət tərəfdaşlığının ləngiməsi
  • Rəqib platformaların meydana çıxması
  • Şəxsi məlumatların qorunması riskləri
  • Maliyyə stabilliyi: qrant və investisiya axtarışı
  • Kiber təhlükəsizlik riskləri
